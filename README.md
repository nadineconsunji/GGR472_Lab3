## GGR472 Lab 3

## Repository Contents

- `index.html`: HTML file to render the main webpage, from which you can access secondary webpages (`BikeLanes.html` contains the webmap)
- `style.css`: CSS file for positioning the map interface and map elements
- `script.js`: JavaScript file containing code for adding elements to the map

#### Purpose 
Lab 3 aimed to build off of Lab 1 and 2's work and develop our knowledge and skills for future application in our group projects. The goal was to achieve more interactivity and classify data from some layers on the map.  

#### The Map
Building off of work generated in Lab 2, this project added a layer that classified the number of bike share stations per neighbourhood in Toronto, a legend for this, pop-up windows naming Toronto neighbourhoods when clicked on, ways to highlight only neighbourhoods near Toronto, changing point size, and general map controls (such as zooming in and making the map full screen). 

This repository contains code for the webpage which includes a web map (with a map style I created) and GeoJSON data layers. 

#### The Website 
Ultimately, while fairly simple, this website contains code to showcase transit near UofT's campus. This includes major roads near campus, bike lanes and parking locations, major TTC stations near campus, and the availability of Bike Share stations across Toronto and near campus. 

#### Files and Data Sources
Raw GEOJson data files can be found in the "Data" folder. These were sourced from both myself and the City of Toronto's Open Data Portal. Image files can be found in the "Images" folder and were sourced from Google searches of each corresponding location. 

Index.html is the main/home webpage, with the titles of other html files being associated with their name on the navigation bar from "index.html". 

The final webpage can be found here: [https://nadineconsunji.github.io/GGR472_Lab3/](https://nadineconsunji.github.io/GGR472_Lab3/)
